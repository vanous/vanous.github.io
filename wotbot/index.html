WotBot

This is the home of WotBot, your WoT Blitz stats and quiz helper.

Previous Next

To get WotBot to your Discord server, click this invite link: here
WotBot is fully open, see source code here: https://github.com/vanous/wotbot. You can use the code for education purposes etc.
See EU Forum discussion http://forum.wotblitz.eu/index.php?showtopic=45862

Command prefix is question mark: ?, for example: ?help

Player/Clan names can be defined also as player@region, for example: The_Honour@eu, or: PRAMO@na
If region (eu, ru, na, asia) is not specified, players and clans are searched in order: eu, ru, na, asia.

Admins: look at conf to see possible server configuration, you can set default's for your server: region, platoon-connector channel or for example the prefix, which is by default question mark: ?
Players: use conf playername to set nick alias, for example ?conf playername The_BLUESTEEL@eu
